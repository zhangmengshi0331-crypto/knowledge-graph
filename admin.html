🔌 Agent API 接口

以下 API 可供 Agent 调用，用于查询和修改数据库。Base URL: http://localhost:5000/api

GET /api/nodes

获取所有节点，或通过 ?keyword=xxx 搜索

curl http://localhost:5000/api/nodes?keyword=AI
GET /api/nodes/:id

获取单个节点详情

curl http://localhost:5000/api/nodes/1
POST /api/nodes

创建新节点

curl -X POST http://localhost:5000/api/nodes \ -H "Content-Type: application/json" \ -d '{"name":"新节点","category":"新能源","node_type":"product","description":"描述..."}'
PUT /api/nodes/:id

更新节点

curl -X PUT http://localhost:5000/api/nodes/1 \ -H "Content-Type: application/json" \ -d '{"description":"更新后的描述","upstream":["原材料A"]}'
DELETE /api/nodes/:id

删除节点

curl -X DELETE http://localhost:5000/api/nodes/1
GET /api/graph?keyword=xxx

获取图谱数据（上下各5层）

curl http://localhost:5000/api/graph?keyword=锂电池
GET /api/fields

获取字段配置列表

POST /api/fields

添加新字段

curl -X POST http://localhost:5000/api/fields \ -H "Content-Type: application/json" \ -d '{"name":"region","label":"地区","field_type":"text"}'
GET /api/logs

查看搜索日志（包含未找到的关键词）

GET /api/stats
